Code Example - PHP front


require '../app/App.php';

$app = new App();

$app->get('/', function() use ($app){
    echo $app->render('index');
});

$app->get('/hello/(:name)', function($name = 'world') use ($app){
    echo $app->render('world', array('name' => $name));
});

$app->run();

Slim Framework

The Slim Framework is a simple yet powerful PHP 5 framework to create RESTful web applications. The Slim micro framework is everything you need and nothing you don't.

Twitter Bootstrap

Twitter Bootstrap, from Twitter. Simple and flexible HTML, CSS, and Javascript for popular user interface components and interactions.

SASS

Sass makes CSS fun again. Sass is an extension of CSS3, adding nested rules, variables, mixins, selector inheritance, and more. Slimstart combines SASS and Bootstrap using Bootstrap-SASS.

Mustache

Mustache. Logic-less templates. We use them both client-side using JavaScript and server-side using Mustache for PHP

CoffeeScript

CoffeeScript is a little language that compiles into JavaScript. CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way. You can use it in Slimstart.

Bourbon.io

Bourbon.io is a simple and lightweight mixin library for Sass. It is bundled in Slimstart.

Grunt

Grunt, the JavaScript Task Runner helps with boring repetitive stuff. In one word: automation. The less work you have to do when performing repetitive tasks the easier your job becomes.

Composer

Composer is a Dependency Manager for PHP. It allows you to declare the dependent libraries your project needs and it will install them in your project for you.
